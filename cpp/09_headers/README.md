# Lesson 09: Headers

In this lesson, we split C++ code into a header file and source files.

The header file tells C++ what exists.  
The source file contains the actual code.

```text
.hpp  declaration
.cpp  definition
```

## Files

- `main.cpp`
- `four_vector.hpp`
- `four_vector.cpp`
- `task_solution_main.cpp`
- `task_solution_four_vector.hpp`
- `task_solution_four_vector.cpp`
